
# 🔗 Link-Orbi

A modern **Link-in-Bio / Developer Profile Aggregator** built with Next.js.  
Create a single customizable page to showcase all your important links, social profiles, and projects.

---

## 🚀 Live Demo
👉 https://link-orbi.vercel.app/

---

## ✨ Features

- 🔗 Add unlimited links (GitHub, LinkedIn, Portfolio, etc.)
- 🎨 Clean and minimal UI
- ⚡ Fast performance with Next.js
- 📱 Fully responsive design
- 🌙 Modern design (can be extended with dark mode)
- 🔐 Easy authentication (if implemented)
- 🧩 Scalable architecture

---

## 🛠️ Tech Stack

- **Frontend:** Next.js, React.js
- **Styling:** Tailwind CSS
- **Backend:** (Optional – Node.js / API routes)
- **Deployment:** Vercel

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/link-orbi.git
cd link-orbi
2️⃣ Install dependencies
npm install
3️⃣ Run development server
npm run dev

App will run on:


http://localhost:3000

🌐 Environment Variables (if needed)

Create a .env.local file:


NEXT_PUBLIC_APP_URL=http://localhost:3000
